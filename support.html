见证者教练是一款面向教练学习者的阅读、练习记录与知识卡片应用，支持 iOS 与 Android 双平台。如果你在使用过程中遇到问题，欢迎通过本页提交反馈；Android 用户也可在「下载与安装」一节获取最新 APK。

下载 Android APK 提交问题反馈 隐私政策

1. 下载与安装

iOS 用户请在 App Store 搜索「见证者教练」下载。Android 用户当前为公开测试版本，通过下面的按钮直接下载 APK 安装包：

下载 Android APK（v1.0.0） 下载 ZIP（IM 兜底）
  • 最低系统要求：Android 8.0（API 26），APK 约 19 MB
  • vivo / 荣耀 / 小米 / OPPO 等国产 ROM 首次安装时可能弹出安全提示，按系统引导放行即可
  • 部分浏览器或 IM 会过滤 APK 后缀，下载受阻时改用 ZIP，解压后再安装
  • 所有历史版本与更新日志：Releases 页面

2. 支持范围

你可以反馈以下类型的问题：

  • App 无法打开、闪退或显示异常
  • 阅读内容、目录、图片或知识卡片显示问题
  • 练习记录、学习进度等本地数据相关问题
  • iOS App Store / Android APK 下载、安装或更新后的使用问题
  • 对产品体验、内容组织或功能改进的建议

3. 如何提交反馈

请通过 GitHub Issues 提交问题反馈： https://github.com/Coldplay-now/iseeyou-privacy/issues

为了便于定位问题，建议在反馈中包含：

  • 设备型号和系统版本（iOS / iPadOS / Android）
  • App 版本号
  • 问题出现的页面或操作步骤
  • 相关截图或录屏

4. 数据与隐私

当前版本的见证者教练采用本地优先设计，阅读进度和练习记录主要保存在你的设备本地。更多说明请查看 隐私政策。

5. 常见问题

我的练习记录会上传到服务器吗？

当前版本不会将练习记录上传到服务器，相关数据仅用于在 App 内展示你的练习和学习进度。

删除 App 后本地数据还会保留吗？

删除 App 通常会移除保存在设备本地的数据。建议在删除前确认是否仍需要保留练习记录。

我可以反馈内容错误或错别字吗？

可以。请通过问题反馈入口说明所在章节、页面或截图，我们会在后续版本中整理修正。

Android 版本和 iOS 版本功能一致吗？

核心功能（阅读、笔记、卡片、练习、分享）保持一致；少量交互按 Android 平台习惯做了适配，例如阅读详情的段落操作改为长按出菜单，整体阅读支持轻触屏幕切换顶栏的「Zen 模式」。

6. 响应时间

我们会尽量及时查看并处理反馈。由于当前应用仍处于早期版本，部分问题可能会安排在后续版本中统一修复。
